# Agent Working Procedure

1. 에이전트는 `## 요구사항` 을 구체화하기 위해 필요한 질문들을 사용자와 나눕니다. 질문은 에이전트가 제안하는 선택지가 포함되어 있습니다.
2. 사용자로부터 받은 답변을 통해 에이전트는 `## 요구사항`을 구체적으로 수정합니다. 요구사항에는 정제된 요구사항만이 담기며 발생한 의사결정은 `## 의사결정 히스토리`에 간략하게 더하는 방식으로 기록합니다.
3. 더이상 질문과 답변할 것이 없을 때까지 1, 2 과정을 반복합니다.
4. 3이 끝나면 에이전트는 사용자에게 `## 요구사항` 리뷰를 요청합니다. 사용자가 승인하지 않으면 다시 앞으로 돌아갑니다.
5. 사용자가 `## 요구사항` 리뷰를 승인하면 에이전트는 `## 작업계획`을 작성합니다.
6. 에이전트는 `## 작업계획` 리뷰를 사용자에게 요청합니다. 사용자가 승인하지 않으면 에이전트는 반영후 다시 리뷰를 요청합니다.
7. 사용자가 `## 작업계획` 리뷰를 승인하면 에이전트는 작업을 진행합니다.
8. 만약 에이전트가 작업중에 `## 요구사항` 과 `## 의사결정 히스토리`에 대해 재검토가 필요하면 즉시 작업을 중단하고 앞으로 돌아갑니다.
9. 에이전트가 모든 작업을 마쳤으면 `## 작업결과`를 작성하고 리뷰를 사용자에게 요청합니다.
10. 사용자는 `## 작업결과` 리뷰를 하고 이를 에이전트가 반영합니다. `## 요구사항` 과 `## 의사결정 히스토리` 에 반영할 사항이 있으면 반영합니다. 
11. 각 작성물은 KST 기준으로 시간을 기록합니다.

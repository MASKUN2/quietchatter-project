# Project Local Gemini Configuration

이 폴더는 QuietChatter 프로젝트 전용 Gemini CLI 설정 및 기술(Skills)을 관리하는 공간입니다.

## 폴더 구조
- skills/: 프로젝트 전용 커스텀 기술(SKILL.md) 정의 파일들이 위치합니다.
- config.json: (필요 시) 프로젝트별 Gemini CLI 동작 설정을 관리합니다.

## 사용 방법
에이전트에게 이 프로젝트에 특화된 반복적인 작업 절차나 도메인 지식을 가르치고 싶을 때, skills/ 폴더 내에 새로운 SKILL.md 파일을 생성하고 지시하십시오.
